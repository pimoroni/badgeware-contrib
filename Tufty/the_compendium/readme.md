# Welcome to the Compendium!

This is... a little fever dream initially intended to test out some raycasting. It was made for the shipping firmware for Badger, and then I decided to port it for Tufty. I hope you enjoy it, and if you can't do that I hope you tolerate it. It supports the QwST Pad automagically, just plug it in and you're good.

Dive into the code a little and you might be able to see how it's put together. There's actually a full adventure game engine under there buried in all the jank, and this demo does barely anything with it - you want to take it and make something a little less nonsensical with it, I'd love to see!
